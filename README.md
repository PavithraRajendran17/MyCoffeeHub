☕ MyCoffeeHub - Full-Stack Coffee Recipe Guide
A premium, data-driven web application designed to explore and learn 100+ professional coffee brewing recipes. Developed as part of the CodeAlpha Internship (Task 1).

🌟 Features
Modern Glassmorphism UI: A sleek, transparent interface inspired by modern design trends.
100+ Variations: Fetches real-time data for various coffee types from a MySQL database.
Surprise Me!: A randomizer tool that picks a unique coffee for you with a single click.
Brewing Timer: An integrated countdown timer to help users time their perfect brew.
Smart Search: Instant filtering to find any coffee recipe in milliseconds.
Responsive Strength Meter: Visual representation of caffeine intensity for each drink.
🛠️ Tech Stack
Frontend: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
Backend: Node.js, Express.js
Database: MySQL
Icons: FontAwesome
📁 Project Structure
index.html - The main entry point for the frontend.
style.css - Custom glassmorphism styling and animations.
script.js - Logic for API calls, timers, and UI interactions.
server.js - Express backend to handle database queries.
coffee_db_backup.sql - The complete database schema and 100+ recipe records.
